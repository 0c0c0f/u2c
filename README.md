## U2C  = Unicode To Chinese


A burpsuite Extender That Convert Unicode To Chinese

Unicode 转中文 的burp suite插件

**version 0.1-0.5:**

前五个版本使用自定义的图形界面，实在原始请求响应的基础上修改数据包，然后进行展示。



**version 0.6：**

完全重写，使用新的tab来展示转码后的响应数据包，不影响原始的响应数据包，更加简洁实用！
